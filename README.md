# 1-Minute Skill App

A simple mobile app built with React Native / Expo.

## Overview

This app helps users improve their skills in just 1 minute a day. It includes:

- Core app logic in the `app/` folder
- Static assets in the `assets/` folder
- Constants and configuration in `constants/`
